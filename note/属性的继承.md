# 属性的继承

## 01. 常见可继承属性
一些 CSS 属性可以向下传递给后代元素。以下是常见的一些可以继承的属性。

### 01.1. 字体
- font
- font-size
- font-family
- font-weight
- font-style
- font-varint

### 01.2. 文本
- text-indent
- text-align
- text-shadow
- line-height
- word-spacing
- letter-spacing
- text-transform
- direction

### 01.3. 元素可见性
- visibility

## 02.常见不能继承属性
- display
- position
- content
- 背景图片、颜色、位置的属性
