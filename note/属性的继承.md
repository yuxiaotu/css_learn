# 属性的继承

## 01.常见可继承属性
一些 CSS 属性可以向下传递给后代元素。

### 字体
- font
- font-size
- font-family
- font-weight
- font-style
- font-varint

### 文本
- text-indent
- text-align
- text-shadow
- line-height
- word-spacing
- letter-spacing
- text-transform
- direction

### 元素可见性
- visibility

## 02.常见不能继承属性
- display
- position
- content
- 背景图片、颜色、位置
